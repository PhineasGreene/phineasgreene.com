Technical Information

This site is made pretty much 100% from scratch. You can check out the source code here:
https://github.com/PhineasGreene/phineasgreene.com.

Front End

The front end is just good old HTML, CSS and Javascript. Content HTML files are slotted into templates with a custom template system.

Back End

This site is hosted on a Vultr machine running Debian 10. The server is made with NodeJS, and uses NGINX as a reverse proxy. Certbot handles HTTPS.
